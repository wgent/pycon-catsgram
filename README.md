# pycon-catsgram
Демо luigi для PyCon Siberia 2016

Luigi:
http://luigi.readthedocs.io/en/stable/

Установка luigi:
```bash
> sudo pip install luigi
```

Запуск standalone scheduler:
```bash
> luigid
```
Итерфейс доступен через: 
http://localhost:8082/

Запуск цепочки вычислений:
```bash
> python main.py
```
